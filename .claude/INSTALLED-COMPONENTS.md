## 🚀 Uso Automático

Los skills se activan **automáticamente** cuando Claude detecta la necesidad. No necesitas invocarlos manualmente a menos que quieras:

### Ejemplos de Activación Automática:
- Pides una feature → `writing-plans` crea el plan
- Encuentras un bug → `systematic-debugging` aplica proceso
- Haces cambios → `test-driven-development` enforcea tests
- Necesitas review → `code-reviewer` agent se activa

### Invocación Manual (Opcional):
```bash
# Si querés forzar un workflow específico
/brainstorm "Nueva funcionalidad X"
/write-plan "Implementar sistema Y"
```
